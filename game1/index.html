Dodge the Saws!

  • Originality: The game I created isn't based off any of the given examples from the Phaser website. I had to think of a way to create an object dodging and object collecting game from "scratch".
  • Technical Merit: Since this is my first digital game, there aren't any highly advanced technical mechanics involved in the game. However, there is a timer/looper which is used to generate the saws endlessly.
  • Prototype Postmortem: I learned the basics of creating an object dodging, platform game. The easiest part included figuring out which assets would be relevant to my game while the hardest part was implementing the correct way to generate saws endlessly.
  • Prototype Assets: All were borrowed from the given sprites page from Phaser or the tutorial page from Phaser.
    • Sprites: Sprites page
    • Tutorial: Tutorial page
  • Prototype Closest Other Game: I used the structure of the "Making Your First Game" tutorial to get a sense of how everything is placed into the game environment.
  • High Concept: Dodge the Saws! is a platform-based game where the player must run around dodging saws while collecting diamonds.
  • Theme: To be honest, there isn't a strong theme, just a dude walking around dodging saws.
  • Mandated Variety: Input: arrow keys to move around and jump
